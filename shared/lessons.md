# Lessons Learned

Agents append to this file when a failure mode repeats. Format:

```
## YYYY-MM-DD — <agent-slug>

**What happened:** <one or two sentences>
**Why it happened:** <root cause>
**Fix applied:** <what was changed>
**Prevent recurrence:** <skill / routine / instruction edit to avoid repeat>
```

Append-only. Do not edit prior entries — append corrections as new entries.
